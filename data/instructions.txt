This package comes with a docker container. All the codes are located at /home/ubuntu/data folder and this is a mounted volume to the host system. 

Steps:

    run the docker compose command
     $ sudo docker-compose up -d 
     
    navigate to http://0.0.0.0:6081/ on a web browser and you will be able to connect to noVNC viewer or open 0.0.0.0:5900 in VNC viewer (eg: Remina) 

    Copy files 
    from  /home/data/gazebo_assets  to  /usr/share/gazebo-11/models
    
    Install extra packages 
    run  install_packages.sh located in data/
